# ❌Tic-Tac-Toe⭕️

Our project can be found [here.](https://tommi267.github.io/tictactoe/)

## Abstract

`Tic-Tac-Toe` will allow users to:
- Play tic tac toe with another player
- Keep track of wins
- Allow user to reset game

## Technologies Used

- Javascript
- HTML
- CSS
